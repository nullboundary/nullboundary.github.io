Pass Ninja 2015

Pass Ninja is a web application I created to generate Apple wallet passes. It was mainly created to explore new web technologies and to advance my skills in deploying web services. The Pass Ninja frontend heavily relies on SVG and the d3.js library. On the server side Pass Ninja uses the Go programming language and is designed and deployed using scalable web technolgies such as docker.
