Mooniverse 2006

MOONIVERSE is an interactive storytelling community for readers and writers. Mooniverse is based on a MOO (Multi-User Dungeon Object Oriented) which is an online text based virtual world that was originally created in 1990. The MOO software has been used to make a non linear interactive multiuser story environment. For this project I learned how to program in the Lambda MOO software environment and worked with my collaborator Makiko Saito to create a interactive multiuser text narrative.
