Work in Progress 2017

This is a solo project being currently developed as a commercial role playing video game for both the PC and VR platforms. The game concept is to create a modern 3d game that incorporates many of the ideas and elements of a traditional rogue-like game. The central element of a rogue like game being high replay-ability through the use of highly randomized procedural world generation. This game uses randomness to generate everything from world layout and texturing, to item placement and enemy stats. The project is being developed using the Unreal Engine 4 and consists of approximately 50% C++ and 50% Blueprint visual programming.
